Smart Dublin — Open Parking Data Review

Dublin Street Parking
Data Review

An independent review of the open parking data published by Dublin's four county councils through Smart Dublin. This analysis examines data completeness, consistency, accuracy, and usability across all four local authorities.

Prepared by Sloto — hello@sloto.ie

Four councils, five files, zero shared standards

We reviewed all publicly available on-street parking data published by Dublin's four county councils through Smart Dublin. Each council publishes their data independently using different formats, structures, and levels of detail.

No shared data standard exists
Each council uses different column names, different file formats, and includes different information. There is no way to combine these datasets without significant manual mapping and standardisation work.

SDCC data overlap — which file to use?

South Dublin County Council publishes two separate files that both contain accessible parking information. This creates a duplication problem with no clear way to reconcile them.

Two files, overlapping data, no shared key
The Paid Parking CSV includes accessible bay counts alongside regular bays for village street parking. The Accessible Parking GeoJSON is a separate dataset of 65 dedicated accessible locations across SDCC. Some on-street village locations appear in both files. Without a shared identifier, there is no reliable way to determine which file is authoritative or how to merge them without double-counting.
